# Projektauftrag: City of Heroes – Lokaler Privatserver (Windows) Richte mir auf diesem Windows-PC einen vollständig funktionsfähigen City of Heroes Privatserver ein, auf dem ich lokal alleine spielen kann. ## Ziel - Ich möchte City of Heroes (i25) komplett auf meinem PC spielen können, ohne Internetverbindung oder fremde Server. - Alle benötigten Dateien kommen in einen Ordner namens COH auf dem Desktop: C:\Users\[Benutzername]\Desktop\COH\ - Am Ende starte ich das Spiel mit einem Doppelklick auf eine Startdatei im COH-Ordner. ## Aufgaben im Detail 1. Software installieren - SQL Server Express (kostenlos, Microsoft) installieren - SQL Server Management Studio (SSMS) installieren - XAMPP installieren (für lokalen Webserver / Account-Portal) - 7-Zip installieren (zum Entpacken) 2. Server-Dateien einrichten (Desktop\COH\) - COH-Server i25 von OuroDev herunterladen (wiki.ourodev.com/I25_Downloads) - COH-xampp.7z herunterladen und entpacken - dbschema.rar herunterladen und Datenbank importieren - i25 Client herunterladen (wiki.ourodev.com/I25_CoH_Client_and_Server) - Ordnerstruktur anlegen: Desktop\COH\Server\ ← Server-Binaries Desktop\COH\Client\ ← Spiel-Client Desktop\COH\XAMPP\ ← Webserver + Account-Portal Desktop\COH\dbschema\ ← Datenbank-Schema 3. Datenbank konfigurieren - SQL Server Express starten, Instanz: SQLEXPRESS - Installationspfad: C:\Microsoft SQL Server (wichtig für DBSchema) - dbschema SQL-Skripte der Reihe nach ausführen 4. Server konfigurieren - In Desktop\COH\Server\bin\ folgende .cfg-Dateien anpassen: servers.cfg, account_server.cfg, auction_server.cfg, chat_server.cfg - Passwort in allen Configs setzen - In servers.cfg eintragen: DefaultAccessLevel = 11 (damit ich als GM/Admin spiele und Charaktere beeinflussen kann) 5. Startskript erstellen Erstelle Desktop\COH\START_SERVER.bat mit folgendem Ablauf: - SQL Server starten (falls nicht aktiv) - XAMPP-Webserver starten - DBServer.exe starten - AuthServer.exe starten - MapServer.exe starten - 10 Sekunden warten - Tequila.exe (Client-Launcher) starten Das Skript soll mit EINEM Doppelklick alles starten. 6. Account anlegen - Account-Portal unter http://127.0.0.1/ aufrufen - Meinen ersten Account erstellen - In Tequila: Manifest-URL http://127.0.0.1/manifest.xml eintragen ## GM-Zugang & Charakterkontrolle Da DefaultAccessLevel = 11 gesetzt ist, habe ich als erster Charakter volle GM-Rechte. Erkläre mir nach der Installation kurz: - Wie öffne ich das GM-Menü? (/mmm im Chat) - Wie setze ich Level, Inf (Geld) und Items per Befehl? - Welche Befehle gibt es zum Spawnen von Gegenständen? ## Systemvoraussetzungen prüfen - Prüfe vorher, ob mindestens 8 GB RAM vorhanden sind - Falls weniger als 32 GB RAM: Windows-Auslagerungsdatei auf 32768 MB setzen - Prüfe, ob genug Festplattenspeicher vorhanden ist (min. 25 GB) ## Quellen / Dokumentation - OuroDev Wiki: wiki.ourodev.com - i25 Downloads: wiki.ourodev.com/I25_Downloads - i25 Server Setup: wiki.ourodev.com/I25_Server_Setup - GM-Befehle: wiki.ourodev.com/I25_Commands - YouTube-Anleitung: youtu.be/XhLgDrk6WiY ## Erwartetes Ergebnis Nach Abschluss soll ich: ✓ Den Server per Doppelklick auf START_SERVER.bat starten können ✓ Einen Superhelden-Charakter erstellen und in Paragon City spielen ✓ Per GM-Befehlen Level, Geld und Items meines Chars steuern können ✓ Vollständig offline und ohne Internetverbindung spielen können

Was wird eingerichtet?
🖥️

Lokaler Server

Der vollständige CoH i25-Server läuft auf deinem PC. Kein Internet nötig, keine externen Server.

🦸

Charakter erstellen

Du erstellst deinen Superhelden im vollständigen Charakter-Editor – Klassen, Kräfte, Kostüm, Name.

⚡

GM-Vollzugriff

DefaultAccessLevel 11: du spielst sofort als Game Master mit Zugriff auf alle Admin-Befehle.

🗂️

Ein Ordner, ein Klick

Alle Dateien in Desktop\COH\ – Start per START_SERVER.bat.

Charakterkontrolle – Übersicht
ℹ️
Mit DefaultAccessLevel = 11 in servers.cfg erhält jeder neue Charakter automatisch volle GM-Rechte. Im Spiel öffnest du das Admin-Menü mit /mmm.
Befehl Kategorie Was er tut
/mmm Admin-Menü Öffnet das vollständige GM-Verwaltungsmenü
/setlevel [1-50] Level Setzt das Level deines Charakters sofort
/give_inf [Betrag] Geld Gibt Influence (CoH-Währung) direkt auf dein Konto
/grantpower [Power-Name] Kräfte Verleiht eine beliebige Kraft/Superpower
/reward [Item-Name] Items Gibt ein bestimmtes Item oder Enhancement
/setaccesslevel [Name] [0-11] Rechte Setzt das Zugriffslevel eines Accounts
/godmode Spiel Unverwundbarkeit an/aus
/fly Bewegung Flugmodus (auch ohne Flug-Kraft)
/teleport [Zone] Bewegung Sofort in eine andere Zone teleportieren
/spawn_critter [Name] Welt Gegner oder NPC in der Welt spawnen
/maxhp Status HP auf Maximum setzen
/fillstatus Status Alle Endurance/HP sofort auffüllen
Vollständige Befehlsliste: wiki.ourodev.com/I25_Commands  ·  homecoming.wiki/wiki/Talk:List_of_Slash_Commands
