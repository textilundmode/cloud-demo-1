# Cloud Demo 1

Ein minimales Demoprojekt mit einem Python-Hello-World-Skript und einer statischen HTML-Seite.

## Inhalt

| Datei | Beschreibung |
|-------|-------------|
| `hello.py` | Python-Skript, das "Hello, World!" auf der Konsole ausgibt |
| `index.html` | Statische HTML-Seite, die im Browser geöffnet werden kann |

## Verwendung

### Python-Skript ausführen

```bash
python hello.py
```

### HTML-Seite öffnen

Öffne `index.html` direkt im Browser oder starte einen lokalen Server:

```bash
python -m http.server 8000
```

Dann im Browser: [http://localhost:8000](http://localhost:8000)
